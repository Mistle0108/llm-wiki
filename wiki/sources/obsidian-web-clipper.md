# Obsidian Web Clipper Help

## Summary

The official help page presents Obsidian Web Clipper as the main browser-based intake tool for bringing web content into an Obsidian vault.

## Key takeaways

- The tool is positioned as a free extension for capturing and annotating web pages.
- Official browser support spans Chrome-family browsers, Firefox, Safari, and Edge.
- The help flow highlights clipping, highlighter mode, reader mode, interpreter prompts, and custom templating.
- The privacy stance is that data is saved locally to the vault, no usage metrics are collected, and the code is open source.

## Implications for this repo

- Web Clipper is a strong default for turning articles into `raw/inbox/` captures.
- It supports the repository thesis of keeping raw captures stable while letting Codex synthesize in `wiki/`.
- If repeated clipping patterns emerge, this vault could standardize them with templates.
