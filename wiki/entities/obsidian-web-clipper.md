# Obsidian Web Clipper

Obsidian Web Clipper is a browser extension that helps move web content into an Obsidian vault.

## Why it matters here

This vault relies on `raw/inbox/` as the intake layer for immutable source captures. Web Clipper is a practical bridge between web reading and that raw-source layer.

## Notable capabilities

- Save web content directly into a vault.
- Highlight passages and selectively capture page elements.
- Use templates, variables, filters, and logic for more structured captures.
- Keep content local to the vault according to the official help page.

## Constraints

- It helps with capture, not synthesis. The maintained knowledge still belongs in `wiki/`.
- Browser extension behavior can vary by browser and site.
