# Introduction to Obsidian Web Clipper

Original source URL: https://obsidian.md/help/web-clipper

## Capture note

This raw capture summarizes the official Obsidian help page for Web Clipper. The canonical source remains the published help page.

## Key points captured from the source

- Obsidian Web Clipper is a free browser extension for saving web content into an Obsidian vault.
- Official browser distribution exists for Chromium browsers, Firefox, Safari, and Edge.
- The help flow emphasizes clipping pages, highlighting passages, reader mode, interpreter prompts, and custom templates.
- The page states that clipped content is saved locally to the vault, that usage metrics are not collected, and that the codebase is open source.
